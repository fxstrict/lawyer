📋 تعليمات الربط — 4 خطوات فقط
  1. افتح script.google.com → مشروع جديد
  2. احذف كل الكود والصق محتوى Code_v4.gs كاملاً → عدّل LAWYER_EMAIL ببريدك
  3. اضغط Deploy → New Deployment → Web App
    Execute as: Me  |  Access: Anyone → انسخ الرابط
  4. ضع الرابط في حقل Apps Script أعلاه → اختبار الاتصال
    ⚡ سيُنشئ الكود جدول البيانات بكل الأوراق والأعمدة تلقائياً!
✅ لا تحتاج لإنشاء جدول بيانات يدوياً — سيُنشأ تلقائياً في Google Drive الخاص بك مع أول اتصال
